## Requirements:

-   Laravel (PHP)
-   Vue.js
-   Bootstrap
-   MySQL

## Task:

-   Create a simple Todo application.

## Features:

-   User authentication (registration, login, logout).
-   Users can create, edit, delete, and mark tasks as complete.
-   Display tasks in a responsive interface using Bootstrap.

## Deliverables:

-   Laravel API backend.
-   Vue.js frontend.
